# Contributing

Thanks for helping improve Gotogo iOS.

## Development

- Keep user-specific Xcode files out of source control.
- Do not commit credentials, provisioning profiles, certificates, private
  backend URLs, or personal network addresses.
- Prefer configuration through Xcode scheme environment variables for local
  development.
- Run focused XCTest coverage for the area you change.

## Pull Requests

- Keep changes scoped and explain user-facing behavior changes.
- Include tests for crypto, storage, messaging, or privacy-sensitive changes.
- Document any new backend endpoint, entitlement, permission prompt, or
  third-party dependency.
